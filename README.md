# Evaluación Final Módulo 4

Esta aplicación permite gestionar usuarios de manera sencilla desde una página web. Puedes ver, agregar o modificar información básica de usuarios usando una interfaz amigable, es parte de la evaluación final del Módulo 4 del curso Fullstack Talento Digital.

## Estructura del proyecto
- **index.html**: Página principal de la aplicación.
- **assets/css/estilos.css**: Hojas de estilo para la aplicación.
- **assets/js/usuarios.js**: Lógica JavaScript para la gestión de usuarios.
- **assets/documents/Links GitHub.txt**: Documentos y enlaces de interés.

## Instrucciones de uso
1. Abre el archivo `index.html` en tu navegador web.
2. La aplicación cargará los estilos y scripts automáticamente.
